CSS3

CSS es un lenguaje de diseño gráfico que permite definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.
Es muy usado para establecer el diseño visual de los documentos web e interfaces de usuario escritas en HTML.

En esta página se irán añadiendo todos los conceptos de CSS vistos en el módulo de Diseño de Interfaces Web

1. CSS Rounded Corners

La propiedad CSS border-radius redondea las esquinas del borde exterior de un elemento. Puedes establecer un solo radio para hacer esquinas circulares, o dos radios para hacer esquinas elípticas

Elemento con bordes redondeados

2. CSS Border Images

La propiedad de CSS border-image permite utilizar una imágen para definir los bordes de los elementos. Esto hace que dibujarlos sea más simple y elimina la necesidad de utilizar muchas cajas en algunos casos.

3. CSS Backgrounds

Background-color es un propiedad de CSS que define el color de fondo de un elemento, puede ser el valor de un color o la palabra "transparente".

CSS backgrounds

Un div con fondo

4. CSS Gradients

El tipo de datos CSS "gradient" indica un tipo de "image" que consiste de una transición progresiva entre dos o más colores.

Gradient Backgrounds

5. CSS Shadows

Con CSS se puede añadir sombra al texto y a los elementos. Mediante text-shadow y box-shadow.
